J'aimerais créer un site static pour un élément d'un escape game. C'est une interface qui immite un ordinateur. Au démarrage, la première page montre un écran d'ordi verrouillé. Il faut un fond noir avec un champ mot de passe sous un une image d'avatar appelé "Utilisateur". Le mot de passe à rentrer est "PUTAINTCON". Il faut afficher un message si ce n'est pas ça. Si le bon code est mis on arrive sur un écran qui simule le bureau d'un ordinateur. Il y a un symbole wifi barré en bas à droite. En cliquand dessus un modal s'ouvre pour demander le code wifi. En même temps il faut lancer le ficher /static/wifi.mp3. Le code wifi est "dorélasol". Le sont est joué en boucle avec une pause de 10 secondes à chaque fois. Si le code est bon on ferme le modal et on change le logo wifi avec un vrai logo wifi plein (non barré).
Sur le bureau de l'ordinateur il y a une appli "Messagerie". L'application est vérouillée par un code qu'il faut rentrer avant de opuvoir l'ouvrir. Le code est 123456. Quand on valide ça ouvre un grand modal qui simule une application de messagerie style whatsapp. Il y a donc à gauche une liste de 4 contacts avec des noms "contact 1", "contact 2", ... Quand on clique sur un contact ça charge à droite un chat qui défile avec des messages. Rempli avec du lorem ipsum. Il y a une bare de teste pour taper un message qui apparaitra dans le chat au dessus.
On peut passer du mode messagerie au mode contact. Le mode contact affiche une liste de contact avec un petit avatar rond et un nom prénom. Quand on clique dessus ça repasse au mode messagerie avec cette personne sélectionnée. Si on clique sur une personne qui n'a pas encore de chat créé, ça en crée un nouveau sans historique de message.

Pour le contact 3, si un message est envoyé il répond "ok j'arrive". Si c'est un autre contact il répond "Tu dis quoi wesh ?". Il faut une animation avec des points qui dit qu'il est en train d'écrire.
Attention pour envoyer un message il faut vérifier que le code wifi a été validé. Sinon on ouvre un modal d'erreur par dessus avec écrit "Connexion indisponible".

Quand l'application "messagerie" est déverouillée il faut lancer un timer en arrière plan de 120 secondes. Après ce temps, il y a une notification qui apparait sur l'écran avec écrit "Hé oublie pas que t'as mis ta clé en G", ce serait dommage de pas la retrouver encore une fois". Le message reste affiché tant qu'on ne clique pas sur la croix pour fermer la notif.

Tous les mots de passes sont sont insensible à la case et aux accents.
